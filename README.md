# Decky Just Juice

Shows the battery icon or percentage in a small manner on the left corner of the display.

This plugin is only about the battery icon, for more customization options for the whole overlay, check out the [MangoPeel](https://github.com/Gawah/MangoPeel) plugin.

![screenshot](assets/screenshot-1.png)
